# LangChain + Gemini Basic Demo

## Overview
This project demonstrates how to use **LangChain** with **Google Gemini** to run a simple query using Python.  
The API key is securely managed using a `.env` file.



## Task Completed
- Installed LangChain library
- Connected to Google Gemini model
- Queried the model using LangChain
- Stored output in README
- Used `.env` file for secure API key handling


## Tech Stack
- Python 3.x
- LangChain
- Google Gemini (Free Tier)
- python-dotenv
